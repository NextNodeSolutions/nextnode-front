# Getting Started with NextNode

Welcome to NextNode! This guide will walk you through setting up your account and deploying your first project.

## Prerequisites

Before you begin, make sure you have:

- A GitHub, GitLab, or Bitbucket account
- Node.js 18+ installed locally (for development)
- Basic familiarity with Git and command-line tools

## Step 1: Create Your Account

1. Visit [nextnode.dev](https://nextnode.dev) and click **Get Started**
2. Sign up using your preferred Git provider
3. Authorize NextNode to access your repositories

## Step 2: Connect Your Repository

Once logged in:

1. Click **New Project** from your dashboard
2. Select the repository you want to deploy
3. NextNode will automatically detect your framework

## Step 3: Configure Build Settings

NextNode auto-detects most settings, but you can customize:

```bash
# Build Command
npm run build

# Output Directory
dist

# Install Command
npm install
```

## Step 4: Deploy

Click **Deploy** and watch your project go live in seconds!
