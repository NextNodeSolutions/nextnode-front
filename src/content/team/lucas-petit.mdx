## About Lucas

Lucas Petit is the creative force behind NextNode's visual identity and user experience. He specializes in designing tools for technical users, making complex functionality feel intuitive and even delightful.

### Design at NextNode

Lucas leads design across all NextNode products:

- **Dashboard UI** - The main control panel for managing deployments
- **Documentation Site** - Interactive, searchable docs with code examples
- **Marketing Website** - Brand identity and conversion optimization
- **Design System** - Component library used across all products

### Design System

Lucas created and maintains NextNode's design system, which includes:

```css
/* NextNode Design Tokens */
:root {
	--nextnode-500: #6366f1; /* Primary brand color */
	--accent-500: #f59e0b; /* Accent for highlights */
	--base-900: #0f0f10; /* Dark backgrounds */
}
```

The design system ensures consistency across all touchpoints while enabling rapid development.

### Design Philosophy

> "Developer tools should respect the intelligence of their users while removing unnecessary friction. Every pixel should earn its place."

Lucas approaches design with a data-informed mindset, regularly conducting user research and A/B testing to validate design decisions.
